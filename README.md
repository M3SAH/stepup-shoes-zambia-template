# Shoe Store Template

A clean, fast, mobile-first **single-page website template** made specifically for shoe shops in Zambia (Kitwe, Lusaka, Ndola, Livingstone, etc.).

Showcase sneakers, formal shoes, school shoes, sandals, boots, gumboots, and more with ZMW pricing, filters (category/type/price), product detail modals, and WhatsApp contact buttons.

## Features

- Fully responsive & mobile-first (most Zambian customers browse on phones)
- Filter shoes by category (Men/Women/Kids), type (Sneakers/Formal/Sandals/Boots/School), and price range
- Clickable product cards → detailed modal with sizes, colors, WhatsApp order link
- New Arrivals / Featured section
- Floating WhatsApp button + contact form (demo mode)
- Very easy to customize — no frameworks, no backend, no build step
- Zambian market focus: ZMW currency, popular styles (Bata school shoes, Nike/Puma-inspired, etc.)

## Technologies

- HTML5
- CSS3 (custom properties/variables)
- Pure Vanilla JavaScript (no dependencies)

## How to Use

1. **Download** or clone this repository
2. Open `index.html` in any browser → works offline
3. **Customize**:
   - Shop name, text, phone number → edit `index.html.`
   - Colors & fonts → edit `: root` variables in `assets/css/style.c.ss`
   - Products → edit the `products` array in `assets/js/script..js`
   - Images → replace Unsplash URLs or add your own photos in `assets/ima.ges/`

### Quick Customization Examples

**Change shop name & hero text**  
→ Open `index.html` → edit `<title>`, `<h1>`, `<p>` in hero

**Add a new shoe product**  
→ Open `assets/js/script.js` → copy-paste one object in `products = [ … ]` and edit values

**Change main color**  
→ Open `assets/css/style.css` → edit `--primary-color` in `:root`
